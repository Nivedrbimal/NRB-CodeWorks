About & GitHub Guide

Quick GitHub Steps (summary)

  1. Create a GitHub account at github.com if you don't have one.
  2. Click New repository, name it MathExercisesWeb, set visibility public, create repo.
  3. Upload files (Add file → Upload files) or push via Git from your PC.
  4. Enable Pages: Settings → Pages → Source → choose main (or master) branch / root → Save.
  5. Your site will appear at https://<your-username>.github.io/MathExercisesWeb/ in a few minutes.

Full step-by-step instructions are included in README.md (download repo and follow that file).
